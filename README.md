# HYGARD Network (Hytale Server)

> Internal README – Game Modes & Systems

HYGARD is a **Hytale-based competitive PVP network** focused on skill-based gameplay, structured progression, and long-term replayability.

---

## Overview

* Platform: **Hytale Multiplayer Server**
* Focus: Competitive PVP + persistent progression
* Structure: Ranked playlists, fast minigames, long-term modes

## Development Standards

* Built around **Hytale combat, physics, and animation systems**
* Uses **instance-based servers** for matchmaking modes
* Persistent modes rely on **world saving & shard architecture**
* Custom kits, items, and abilities use **Hytale asset bundles**

---

## Design Principles

* Skill-first competitive balance
* Shared seasonal systems across ranked modes
* Clear progression and rewards
* Community-driven iteration via playtesting
